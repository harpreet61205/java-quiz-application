# Java Quiz Application

## Description
This project is a Java-based quiz application developed to test user's knowledge of Java programming concepts. The application presents multiple-choice questions and evaluates the user's score based on correct answers.

## Technologies Used
- Java
- NetBeans IDE

## Features
- Multiple-choice quiz questions
- User score calculation
- Simple and interactive interface

## Purpose
This project was developed to practice Java programming and application development concepts.
